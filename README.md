# Rangeway Energy - Modern Test Site

A modern, single-page website for Rangeway Energy, inspired by contemporary tech aesthetics (HyWatts.com).

## Features

- **Modern Design**: Clean, minimalist aesthetic with smooth animations
- **Single-Page Layout**: All content on one scrolling page with smooth navigation
- **Responsive**: Fully responsive design that works on all devices
- **Performance**: Fast loading with optimized assets
- **Jekyll-Powered**: Built with Jekyll for easy GitHub Pages deployment

## Content Sections

1. **Hero** - Where EV Charging Meets Hospitality
2. **Indoor Comfort** - Premium amenities and services
3. **Why Rangeway** - Mission and approach
4. **Baker Basecamp** - Flagship location details
5. **Network** - Expansion corridors
6. **For Travelers** - Amenities and Pathfinder Rewards program
7. **Investors** - Investment opportunities
8. **Statistics** - Market data
9. **Contact** - Get in touch
10. **Newsroom** - Latest updates (dynamically loaded)

## Design Inspiration

Inspired by HyWatts.com with:
- Frosted glass navbar effect on scroll
- Modern typography and spacing
- Smooth scroll animations
- Contemporary card designs
- Gradient accents

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000` to view the site.

## Deployment

This site is designed for GitHub Pages deployment. Simply push to the `main` branch and enable GitHub Pages in repository settings.

## Tech Stack

- Jekyll 4.3+
- Vanilla JavaScript
- CSS3 with CSS Variables
- Responsive Grid Layouts
